# Code Of Conduct

Prompthon Agentic Labs is a public working lab. We want contributions,
critique, and review to stay specific, respectful, and useful for the next
person.

This code of conduct applies in this repository and in project-linked community
spaces used to coordinate lab work.

## Expected behavior

- Critique ideas, structure, evidence, and implementation details rather than
  people.
- Assume good faith first, then be explicit about problems and supporting
  evidence.
- Keep attribution boundaries clear when discussing outside material.
- Give reviewers enough context to evaluate the change without guessing.
- Respect editorial decisions about scope, placement, and repository fit.

## Unacceptable behavior

- Harassment, discrimination, threats, intimidation, or personal attacks.
- Repeated bad-faith argument, thread derailing, or pile-ons after a decision
  has been recorded.
- Posting private information without permission.
- Passing copied upstream prose, screenshots, diagrams, or large code surfaces
  off as repo-native work.
- Spam, unrelated promotion, or non-project recruiting in project spaces.

## Reporting

Do not open public issues for conduct reports.

If you experience or witness a conduct problem, contact the maintainers
privately. If no direct maintainer contact is obvious, use the path described in
`SUPPORT.md` to request a private moderation channel.

## Enforcement

Maintainers may edit, hide, lock, or remove content; ask for changes; restrict
participation; or ban accounts from project spaces when needed to protect the
community and keep collaboration workable.
