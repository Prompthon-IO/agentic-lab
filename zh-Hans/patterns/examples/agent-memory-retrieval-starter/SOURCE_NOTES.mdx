这个入门项目将记忆/检索讨论转化为一个由仓库拥有的小型代码形态，同时不引入提供方配置或框架特定示例。

## 仓库原生更改

- 将示例缩减为可工作的记忆、检索查询和 artifact 提升。
- 增加了一个小型 personal-context import surface，让示例能够区分导入的偏好事实、检索输入与持久化工件。
- 移除了提供方配置、向量存储讲解以及特定包代码。
