# 支持

使用与你在 Prompthon Agentic Labs 中所需帮助类型相匹配的支持路径。

## 在打开任何内容之前

先阅读：

- `README.md`
- `CONTRIBUTING.md`
- `contributor-kit/index.mdx`
- `contributor-kit/github-issue-guide.md`

这些文档定义了仓库结构、贡献类型、模板以及审查预期。

## 在以下情况打开 issue

如果你有以下情况，请创建 GitHub issue：

- 具体的 bug 或损坏的路径
- 实验室内容提案
- 源项目提案
- 仓库功能或流程提案

请使用仓库的 issue 表单，这样请求会以适合审查的正确格式进入流程。

## 在以下情况使用社区讨论

请使用 Prompthon IO Discord 进行：

- 在你还不确定某件事是否应成为 issue 之前的早期提问
- 帮助选择正确的贡献面
- 尚未明确范围的一般讨论

Discord: https://discord.gg/sDE2HhGTg4

## 安全报告

不要就漏洞或敏感配置问题提交公开 issue。请改为遵循 `SECURITY.md`。
