This directory turns the repository into a path-first lab instead of a
single chapter stream.

## Primary guides

- [Explorer](/reading-paths/explorer): for people who want a high-level understanding and a
  recommended set of pages rather than a strict sequence.
- [Practitioner](/reading-paths/practitioner): for people who want to use agent tools and
  skills in real work without becoming professional agent builders.
- [Builder](/reading-paths/builder): for engineers who want a linear path into
  professional agent design and implementation.
- [Contributor](/reading-paths/contributor): for people who want to edit, curate, or
  extend the repository itself.

## Shared supporting guides

- [Environment Setup](/reading-paths/environment-setup): shared local setup and starter
  verification for practitioner, builder, and contributor paths.
- [Sample Projects](/reading-paths/sample-projects): the public index of current
  repo-owned starter projects.
- [Practitioner Skills](/skills): installable Codex skill packages for local-first,
  repeatable workflows.
